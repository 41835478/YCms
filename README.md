## YCms
Free,open-source CMS based on the Laravel PHP Framework


## Requirements

- **PHP : 5.6+
- **Laravel : 5.3+
- **Mysql
- **Redis !!!



## Installation

1. **`git clone https://github.com/iScript/YCms.git`**
1. **`cd YCms`**
1. **`chmod -R 777 storage/`**
1. **`composer install`**
1. **`cp .env.example .env && vim .env`**
1. **`php artisan migrate`**
1. **`php artisan db:seed`**
1. **`php artisan serve`**
